Privacy Policy

How we protect and manage your medical data

Last Updated: June 10, 2023

1. Introduction

Welcome to Certisure Medical ("we," "our," or "us"). We specialize in secure medical data management with strict access controls. This Privacy Policy explains how we collect, use, and protect your sensitive medical information.

Our platform handles the following medical data fields for each record:

Medical Name: Identification of the medical facility or provider
Date: When the medical service was provided
ID: Unique patient identifier
Name: Patient's full name
Passport: Travel document information (when applicable)
Agent: Authorized representative handling the case
Status: Current processing status
Remarks: Additional notes or comments

2. Data Collection & Access

We implement a strict role-based access system where:

  • Agents can only view records associated with their reference code
  • Sensitive fields like passport numbers are partially masked
  • All data access is logged with timestamp and user identification
Unauthorized attempts to access other agents' data will result in immediate account suspension.

3. Security Measures

Encryption

All data is encrypted both in transit (TLS 1.3) and at rest (AES-256)

Authentication

Multi-factor authentication required for all agent accounts

Audit Logs

Complete record of all data access and modifications

Infrastructure

Enterprise-grade servers with regular security patches
